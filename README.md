Realtime Drawing Game built with SocketIO, Gulp and Node

## Install

yarn add @babel/{core,node,preset-env}
yarn add express
yarn add nodemon
yarn add socket.io

```
    "@babel/core": "^7.9.6",
    "@babel/node": "^7.8.7",
    "@babel/preset-env": "^7.9.6",
    "express": "^4.17.1",
    "nodemon": "^2.0.4",
    "socket.io": "^2.3.0"
```

## Babel Setting

.babelrc

```
    {
        "presets": ["@babel/preset-env"]
    }

```
